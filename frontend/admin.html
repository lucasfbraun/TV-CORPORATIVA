💾 Backup e Restauração
📡 Como Acessar as TVs
Link por TV: Cada TV criada gera um link único.
Exemplo: http://[IP]:8080/display.html?tv=recepcao

Servidor local: Execute python server.py e acesse pelo IP da rede.

Atualização: A tela sincroniza a cada 60s. Ao salvar, o sinal é enviado imediatamente (mesma rede).
📊 Credenciais do Grafana
🔒 As credenciais ficam só no servidor. Ao salvar, ele faz login novamente e reinicia a captura.
🔌 Nova Integração
🔒 A senha fica só no servidor — não vai para as TVs nem para o conteúdo público.
🔌 Adicionar Integração à grade
O usuário e a senha ficam no cadastro da integração. Clique em ➕ Nova para criar uma aqui mesmo, ou edite em Sistema → 🔌 Integrações.
⏱️ Sem duração fixa: a integração é uma playlist e alterna o conteúdo sozinha (usa o tempo padrão da grade para passar ao próximo slide).
